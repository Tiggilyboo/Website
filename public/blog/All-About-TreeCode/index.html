All About TreeCode

Feb 27, 2016   TreeCode  Code Editor  Go 

What is TreeCode?

It’s a web-based code editor. What that’s it? Well, the tree part comes in from tracking code file references.

As of writing this post, the project is written using Go or Golang as a root language for hosting a local webserver and serving content. The type of content is nothing out of the ordinary, HTML5 & CSS3, JS, JQuery front end interactions. Sitting on top of this, I’ve selected some libraries to implement; The first of which is the Ace code editor for a js code editor. The code editor has many common features such as syntax highlighting, intellisense-esque predictions and key bindings. I have some uncertainty about the choice of graphing library to render tree structures and leaf nodes representing code files, however this can be solidified once the underlying systems are written.

Requirements

  • User can create a new project
    • User can create a new code file in the project
    • User can save code file, program deciphers code extension in order to parse code references
  • User can load a saved project
    • User can view the project in a tree structure with referenced parent/child relations
    • User can load a saved code file in the project
